sylph-web-gut In-browser gut metagenomic profiling

Multi-sample profile

Drop several FASTQs — sylph profiles each against the loaded gut database and assembles a relative-abundance matrix.

100% browser-based — your files never leave your computer. Computation happens entirely in your browser. No data is sent to any server.
Quick-and-dirty preview. This tool is meant for fast in-browser checks of gut metagenomic composition. It does not achieve the reliability of sylph, MetaPhlAn4, or Meteor2 run natively — use those for real analyses. This is an unofficial adapted WebAssembly port of sylph — not supported or endorsed by the sylph authors. If you use the results, please cite the sylph paper (Shaw & Yu, Nat. Biotechnol. 2024) and the UHGG catalog (Almeida et al., Nat. Biotechnol. 2021).

Reference database

Pick a database and click Load database. First load is the slow part.

Samples

    Recommended: 3,000,000 reads — keeps memory under the 4 GB wasm32 ceiling. Higher values are allowed but the slider turns red past 3 M because the run may fail.
    Above 3,000,000 reads may exceed the 4 GB wasm32 memory ceiling — profiling may fail. Consider lowering the value, or running sylph on a server.
