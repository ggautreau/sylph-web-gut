FASTQ downsampler

Keep the first N reads from a FASTQ (gzipped or plain). Runs locally — the file never leaves your machine.

Quick-and-dirty preview. This tool is meant for fast in-browser sanity checks of gut metagenomic composition. It does not achieve the reliability of sylph, MetaPhlAn4, or Meteor2 run natively — use those for real analyses. This is an unofficial adapted WebAssembly port of sylph — not supported or endorsed by the sylph authors. If you use the results, please cite the sylph paper (Shaw & Yu, Nat. Biotechnol. 2024) and the UHGG catalog (Almeida et al., Nat. Biotechnol. 2021).
